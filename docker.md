## 核心指令 (Core Commands)

### 註冊庫與搜尋 (Registry & Search)

- `docker login`: 登入 Docker Registry (預設為 Docker Hub)。
- `docker logout`: 登出 Docker Registry。
- `docker search <image-name>`: 在 Docker Hub 搜尋映像檔。
