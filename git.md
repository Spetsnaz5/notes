## 1. Setup and Basics (設定與基礎)

### `git init`
建立新的本地端 Repository。
```bash
git init
```

### `git clone`
複製遠端的 Repository 檔案到本地端。
```bash
git clone [Repository URL]
```

### `git remote`
與遠端倉庫建立連接。
```bash
# 新增遠端
git remote add <remote-name> <Repository URL>

# 查看遠端
git remote -v
```
